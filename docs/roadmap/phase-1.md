# Phase 1 Roadmap (6-8 Weeks)

## Objective

Ship `devguard-ai-validator` MVP on top of `devguard-core`.

## Milestones

1. Week 1-2: core scanner, 5 checks, JSON/SARIF output
2. Week 3-4: CI action, quality baseline, fixture corpus
3. Week 5-6: false-positive reduction and confidence tuning
4. Week 7-8: package publishing and community feedback cycle

## Exit Criteria

1. Runs on Python 3.10+
2. Produces deterministic output on sample corpus
3. Integrates into CI with non-zero exit on findings
4. Includes docs and contribution guide
